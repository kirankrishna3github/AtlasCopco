form frm_kpi_c_004
using ip_org_level type yam_kpi_org_level
changing ep_wert ep_waers.

  data: lv_date like sy-datum.

  gv_prevdat = sy-datlo - 1.

  lv_date = sy-datum.
  move '01' to lv_date+6(2).
  move '01' to lv_date+4(2).
  gv_begyr = lv_date.

  IF gt_invy_list[] IS INITIAL.
    SELECT *
    INTO TABLE gt_invy_list
    FROM vbrk
    WHERE vkorg EQ ip_org_level-vkorg
      AND erdat BETWEEN gv_begyr AND gv_prevdat.
  ENDIF.

  LOOP AT gt_invy_list.
*.... Set value for credit memo, cancellation to negative
    IF gt_invy_list-vbtyp CA 'NO'.
      gt_invy_list-netwr = gt_invy_list-netwr * -1.
    ENDIF.

    ADD gt_invy_list-netwr TO ep_wert.

  ENDLOOP.

*.. Current Assumption is no currency conversion is needed/everything
*.. billed in same currency.
  ep_waers = gt_invy_list-waerk.

endform.
